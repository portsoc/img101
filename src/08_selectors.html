Animations can make a page feel more responsive, and avoid jumpiness - but - you have to be careful with your CSS selectors.

  1. Observe the :hover pseudo selector on the .exciting class. When the image is hovered over, it flips end-over-end and grows in size. However, as it flips it becomes thinner until the mouse is no longer hovering over it, at which point the rotation begins to undo. This is not what we want.
  2. To solve this change the .exciting:hover selector so that it reads p:hover .exciting. This is saying that the declarations in this CSS rule will apply to an .exciting class which has an ancestor p element that is being hovered over.
  3. Move the mouse anywhere over the paragraph and the image will flip smoothly.
