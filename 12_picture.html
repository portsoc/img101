The picture element can be used to create responsive images which choose an appropriate image basedon the size of the containing area. In this case the container is the body, and the image is set to fill the width of the body (100vw). The bitmap that gets chosen by the browser depends on the size of the window.

NB: The img element is required. The picture element provides the img element with a rich context and new mechanisms for selecting source images that are presented therein. This is why when we want to style the image, we can still style img directly (rather than the picture element).
