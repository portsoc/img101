  • Observe the figure element which may contain an image and a figcaption element.
  • A figure can contain more than just img elements, for example, a table of data in an academic paper usually has a caption.
  • figure elements may be styled with border, margins, colours, etc., like any other elements.
  • In this example we use the flex display property. This allows us to change the orientation of the image and text with the flex-direction property.
  • The two @media queries provide different layouts when the page is wide (landscape orientation) or narrow (portrait orientation).
